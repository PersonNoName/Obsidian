# FreeTrader 后端项目概述

## 项目简介

FreeTrader 是一个 ETF（交易所交易基金）数据跟踪与投资分析平台的后端服务。该项目提供了 ETF 板块分类、净值数据查询、用户收藏管理等功能。

## 项目基本信息

| 属性 | 值 |
|------|-----|
| 项目名称 | FreeTrader Backend |
| 项目版本 | 0.0.1-SNAPSHOT |
| 包名 | com.freetrader |
| Java 版本 | 17 |
| 框架版本 | Spring Boot 3.2.1 |

## 核心功能模块

1. **用户认证模块** - 注册、登录、JWT 认证
2. **ETF 板块管理** - 板块列表、板块详情、收益率计算
3. **用户收藏功能** - 收藏/取消收藏板块
4. **交易日历** - 支持交易日查询

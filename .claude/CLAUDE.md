# Obsidian 笔记系统

这是我的技术学习笔记库，涵盖前后端开发和游戏开发。

## 笔记结构

```
Obsidian/
├── 01-前端开发/          # HTML, CSS, JavaScript, TypeScript
│   ├── React/
│   ├── Vue/
│   └── 框架技巧/
├── 02-后端开发/          # Node.js, Python, Go 等
│   ├── 数据库/
│   └── API设计/
├── 03-游戏开发/          # Unity, Godot, WebGL
│   ├── Unity-CSharp/
│   ├── Godot-GDScript/
│   └── 游戏算法/
├── 04-通用技能/          # Git, Docker, Linux
├── 05-项目实践/          # 实战项目记录
└── 06-学习笔记/          # 教程和文档整理
```

## 技术栈

**前端**: JavaScript, TypeScript, React, Vue, TailwindCSS, HTML/CSS

**后端**: Node.js, Express, PostgreSQL, Redis, RESTful API

**游戏开发**: Unity (C#), Godot (GDScript), Three.js

**工具**: Git, Docker, VS Code, Figma

## 代码规范

- 编程语言使用对应标准风格（JS 用 camelCase，Python 用 snake_case）
- 代码块标注语言类型
- 复杂逻辑添加注释说明

## 学习习惯

- 记录概念时用「是什么 → 怎么用 → 为什么」的格式
- 重要知识点打上对应标签（如 #react #unity #算法）
- 每个技术建立「速查表」便于复习
- 定期整理：将零散笔记合并到体系化文档

## 常用操作

- `Ctrl + O` - 快速搜索笔记
- `Ctrl + Shift + F` - 全局搜索标签
- `Ctrl + P` - 命令面板
